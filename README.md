# AI Customer Support Chatbot SaaS

Stack:

Frontend
- Vite
- React
- Tailwind
- Lucide Icons

Backend
- NestJS
- PostgreSQL
- Prisma

Monorepo
- PNPM Workspaces
- Turborepo

Apps
- web → SaaS dashboard
- widget → embeddable chat widget
- api → backend API

Run locally:

pnpm install
docker compose up -d
pnpm dev
